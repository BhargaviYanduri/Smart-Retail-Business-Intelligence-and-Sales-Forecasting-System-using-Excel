# Sales & Inventory Management Dashboard 📊

## Overview
This project provides a comprehensive dataset and analysis framework for managing **sales targets, orders, inventory, products, customers, and regional performance**. It is designed to help businesses track progress against sales goals, monitor inventory levels, analyze customer segments, and forecast future sales.

## Features
- **Sales Target Tracking**  
  Monthly sales targets vs. actual achievements for 2025.

- **Order Management**  
  Detailed order records including customer, product, quantity, sales, profit, and margins.

- **Inventory Control**  
  Current stock levels, reorder alerts, and dead stock identification.

- **Product Catalog**  
  Product details with categories and cost prices.

- **Customer Insights**  
  Customer demographics, segmentation, and regional mapping.

- **Regional Analysis**  
  State-to-region mapping for performance tracking.

- **Forecasting**  
  Sales forecasting with confidence bounds for future planning.

- **KPIs Dashboard**  
  Key performance indicators such as:
  - Total Sales
  - Total Profit
  - Profit Margin %
  - Average Order Value
  - Customer Retention Rate
  - Inventory Turnover
  - Target Achievement %

## File Structure
- **Sales Target Table** – Monthly targets for 2025  
- **Order Table** – Transaction-level sales and profit data  
- **Inventory Table** – Stock levels and reorder alerts  
- **Product Table** – Product details with categories and costs  
- **Customer Table** – Customer demographics and segments  
- **Region Table** – Mapping of states to regions  
- **Sheet1 & Sheet3** – Aggregated KPIs, forecasts, and pivot analysis  
- **Dashboards (dash1–dash5)** – Visual summaries of performance metrics
